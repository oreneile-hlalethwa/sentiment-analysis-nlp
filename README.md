# Movie Review Sentiment Analysis

An NLP machine learning project that classifies movie reviews as positive, negative, or neutral.

## Results

| Method | Accuracy |
|--------|----------|
| Rule-Based | 23.60% |
| Basic ML (BoW) | 63.39% |
| Tuned ML (BoW) | 65.11% |
| TF-IDF | 66.02% |

## Tools Used

- Python
- Pandas
- Scikit-learn
- NLTK
- Matplotlib
- Jupyter Notebook

## Files

- `sentiment_analysis.py` — Main ML pipeline
- `dashboard.ipynb` — Visualization dashboard
- `exports/` — Exported data files
